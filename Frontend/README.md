📚 Book Store App

A simple full-stack Book Store application where users can browse, add, update, and delete books.

🚀 Features
View all books
Add new books
Edit book details
Delete books
Search functionality (optional)
🛠️ Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
📂 Project Structure
book-store/
├── client/      # Frontend (React)
├── server/      # Backend (Node + Express)
└── README.md
⚙️ Installation
# Clone the repo
git clone https://github.com/your-username/book-store.git

# Go to project folder
cd book-store

# Install backend dependencies
cd server
npm install

# Install frontend dependencies
cd ../client
npm install
▶️ Run the App
# Run backend
cd server
npm start

# Run frontend
cd client
npm start
🌐 Environment Variables

Create a .env file in /server:

MONGO_URI=your_mongodb_connection_string
PORT=5000
